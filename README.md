# event-chatbot-api-ts

Typescript API for Recast AI (SAP Conversational AI) powered bots.

## Technologies used

- Typescript
- Recast AI
- Serverless
- AWS Lambda
- AWS DynamoDB
- AWS Translate

## How to run

- Make sure you have AWS account
- Do AWS config
- You will need also to create DynamoDB
- call `sls deploy` to deploy
